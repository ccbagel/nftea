# This is a NFT marketplace app built using Nextjs.

## Features:

- Static & Server-side-rendered pages
- Incremental static regeneration
- Image optimization
- Tailwind utility classes styling
- Custom React hooks
- Built in API

### Static pages:
- Homepage
- login page
- signup page

### SSR pages:
- Dashboard

### Custom hooks:
- Getting current user, wallet, and balance for tx
